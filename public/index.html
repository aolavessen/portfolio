Kids Chores

Live Code

Kids Chores is my final project at The IronYard. I teamed up with another developer for a three week project. The idea was to create an interactive site that both parents and kids could use to track chores around the house. It is set up to have seperate logins for both parents and children. Once a parent assigns a chore it shows on the childs site. Once the child does the chore and clicks a box the parent is notified to check. If the chore is done successfully then a child will earn the points associated. Then the child can visit their onsite store to choose from a list of prizes. It is fully customizable by the parent for chores, points and prizes. We worked on both the site and the server. We used HTML, SASS, JavaScript, Ember.js and Materialize for the front end build.
